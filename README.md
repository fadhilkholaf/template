# Template

Fadhilkholaf next.js fullstack app starter template

## How to start

### Initialize Template

```bash
pnpx create-next-app@latest --example https://github.com/fadhilkholaf/template template
```

### Go To Project Directory

```bash
cd template
```

### Install Dependency

```bash
pnpm install
```

### Setup .env file

### Run the server

```bash
pnpm dev
```

## Features

- Next.js 15.1.7
- React 19
- Tailwind CSS 4
- Prisma ORM
- Auth.js
- Optimized SEO
