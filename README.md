# Template

Fadhilkholaf next.js starter template

## How to start

### Initialize Template

```bash
pnpx create-next-app@latest --example https://github.com/fadhilkholaf/template template
```

### Go To Project Directory

```bash
cd template
```

### Install Dependency

```bash
pnpm install
```

### Run the server

```bash
pnpm dev
```
